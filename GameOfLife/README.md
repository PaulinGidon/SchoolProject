# **Jeu de la Vie – John Conway**

Ce projet simule le classique *Jeu de la Vie* de John Conway.  
Avant de compiler, il faut avoir téléchargé la bibliothèque `ncurses`. Pour l’installer, utilisez :

```bash
sudo apt update
sudo apt install libncurses5-dev libncursesw5-dev
```

Pour exécuter le programme, utilisez la commande `make` dans un terminal Linux. Cela compile tout le projet.  
Ensuite, lancez le jeu avec :

```
./JDLV
```

Une fois à l'intérieur du jeu, suivez simplement les instructions à l'écran. Toutes les commandes et modes de lancement sont détaillés dans le programme.

## **Présentation du Jeu de la Vie**

Le Jeu de la Vie est un automate cellulaire en 2D où chaque cellule est soit **vivante**, soit **morte**.  
L'évolution se fait par étapes—appelées *générations*—selon trois règles simples :

### **Survie**
Une cellule vivante avec **2 ou 3** voisines vivantes reste vivante.

### **Mort**
- **Mort par isolement :** Une cellule vivante avec **moins de 2** voisines vivantes meurt.  
- **Mort par surpopulation :** Une cellule vivante avec **plus de 3** voisines vivantes meurt.

### **Naissance**
Une cellule morte avec **exactement 3** voisines vivantes devient vivante.

Chaque cellule ne regarde que ses **8 voisines** (haut, bas, gauche, droite et diagonales).

## **Commandes Make**

| Commande      | Description |

| **make**     | Compile le projet et crée l'exécutable **JDLV** |
| **make clean** | Supprime les fichiers `.o` générés par make |

## **Modes de lancement du jeu**

| Option | Description |
|--------|-------------|
| **1** | Lance une grille aléatoire avec des cellules vivantes placées aléatoirement |
| **2** | Lance une grille prédéfinie à partir du fichier `motif.txt` (par défaut : un simple planeur) |
| **3** | Lance une grille contenant un canon à planeurs |

## **Commandes dans le jeu**

| Touche | Action |
|--------|--------|

| **z** | Réduit le délai entre deux générations |
| **e** | Augmente le délai entre deux générations |
| **s** | Passe une génération et désactive le mode automatique |
| **q** | Quitte le jeu |

# **Game of Life – John Conway (English)**

This project simulates John Conway's classic *Game of Life*.
Before compiling, you need to have the `ncurses` library installed. To install it, use:

```bash
sudo apt update
sudo apt install libncurses5-dev libncursesw5-dev
```

To run the program, execute the command `make` in a Linux terminal. This compiles the entire project.
Then launch the game with:

```
./JDLV
```

Once inside the game, simply follow the on-screen instructions. All commands and launch modes are detailed within the program.

## **Game of Life – Overview**

The Game of Life is a 2D cellular automaton in which each cell is either **alive** or **dead**.
The evolution occurs in steps—called *generations*—based on three simple rules:

### **Survival**
A living cell with **2 or 3** living neighbors stays alive.

### **Death**
- **Death by isolation:** A living cell with **fewer than 2** living neighbors dies.
- **Death by overpopulation:** A living cell with **more than 3** living neighbors dies.

### **Birth**
A dead cell with **exactly 3** living neighbors becomes alive.

Each cell only considers its **8 neighbors** (up, down, left, right, and diagonals).

## **Make Commands**

| Command      | Description |

## **Game Launch Modes**

| Option | Description |
|--------|-------------|
| **1** | Launches a random grid with randomly placed live cells |
| **2** | Launches a predefined grid using the file `motif.txt` (default: a simple glider) |

| **q** | Quits the game |

## **In-Game Commands**

| Key | Action |
|-----|--------|
| **a** | Enables automatic mode (continuous generations) |
| **z** | Decreases the delay between two generations |
| **e** | Increases the delay between two generations |
| **s** | Advances the game by one generation and disables automatic mode |
| **q** | Quits the game |
